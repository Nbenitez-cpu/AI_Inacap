# AI_Inacap

## Sistema de Recomendación

Este proyecto contiene un sistema simple de recomendación de películas desarrollado con GitHub Copilot.

## Pasos seguidos:

1. Creación de cuenta de estudiante en GitHub

2. Instalación de Git y Python

3. Configuración de GitHub Copilot en VS Code

4. Generación del código con Copilot

5. Commit y push a GitHub

## Código

El archivo recommendation\_system.py permite recomendar películas según el género seleccionado por el usuario.
