## Sistema de Recomendación

Este proyecto contiene un sistema simple de recomendación de películas desarrollado con GitHub Copilot.

## Pasos seguidos:

1. Creación de cuenta de estudiante en GitHub

Nos dirigimos a https://github.com/education/students?locale=es-419 para la creacion de nuestra cuenta como estudiantes y seleccionamos la cuenta gratuita.


2. Creacion de repositorio

Nos dirigimos al apartado "NEW" dentro del apartado repositorios y creamos uno nuevo.

3. Configuración de GitHub Copilot en VS Code

Ingresamos a VS Code y nos dirigimos al el apartado extensiones para poder instalar python el cual nos servira como libreria y nos permitirar ejecutar el codigo y comandos. Luego buscaremos GitHub Copilot y lo instalaremos el cual nos servira para agregar el codigo de manera mas rapida y sencilla.

4. Generación del código con Copilot

Una vez instalada la extension bastara con solo pedirle el codigo que queramos agregar y el nos ayudara de manera rapida y sencilla.
5. Commit y push a GitHub

una vez terminado cualquier configuracion o modificacion dentro de Visual Studio deberemos guardar utilizando la linea terminal del mismo programa utilizando los siguente comandos:


- git add nombredelarchivo :para agregar el archivo
- git status :paraverificar si el archivo y su status
- git commit -m para subirlon y agregarle un mensaje descriptivo dentro de GitHub
- git push para publicarlo en el repositorio

## Código

El archivo recommendation\_system.py permite recomendar películas según el género seleccionado por el usuario.
